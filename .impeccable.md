# Design Context

## Users
Personal users managing daily tasks and goals. They value focus, clarity, and efficiency over decorative flourishes. The interface should feel like a trusted tool - reliable, fast, and unobtrusive.

**Jobs to be done:**
- Quickly capture tasks without friction
- Review and prioritize work
- Track progress with visual feedback
- Maintain overview without cognitive overload

## Brand Personality
**Professional, Refined, Thoughtful**

The interface should feel like a well-crafted writing instrument - precise, elegant, and disappearing into the background so users can focus on their work. Not sterile or cold, but deliberately restrained. Every pixel serves a purpose.

**Emotional goals:** Calm confidence, clarity, satisfaction from completion

## Aesthetic Direction
Inspired by Bear and Craft: typography-driven design with subtle animations, generous whitespace, and meticulous attention to detail. No decorative elements that don't serve function.

**Visual tone:** Editorial elegance meets productivity utility

**Theme:** Dual themes (light/dark) with smooth transitions

**Anti-references:** 
- Generic admin dashboard aesthetics
- Overuse of shadows and gradients  
- Playful/cartoonish styling
- Cluttered interfaces with excessive UI chrome

## Design Principles

1. **Typography First**: Type carries the visual weight. Use a refined serif for headings paired with a clean sans-serif for body text. High contrast in scale creates clear hierarchy.

2. **Subtle Motion**: Animations should feel physical and purposeful - items slide, fade, or scale with exponential easing. Nothing bounces or springs. Respect reduced motion preferences.

3. **Restraint as Luxury**: Generous spacing, limited color palette, minimal borders. The absence of visual noise is a premium feature.

4. **Progressive Disclosure**: Show what's needed now, reveal sophistication through interaction. Hover states expose secondary actions without cluttering the primary view.

5. **Theme Cohesion**: Light and dark themes are equally considered, not afterthoughts. Colors use OKLCH for perceptual consistency across both modes.

## Typography Direction
- **Display**: A refined serif with character (e.g., EB Garamond, Noto Serif Display)
- **Body**: A neutral sans-serif with excellent readability (e.g., Inter, system fonts)
- **Scale**: 5-step modular scale with 1.25 ratio for clear hierarchy

## Color Direction
- **Primary accent**: Single purposeful color (deep blue or charcoal) used sparingly
- **Neutrals**: Tinted toward brand hue, never pure gray
- **Semantic colors**: Muted red/orange/green for status - visible but not jarring
- **Use OKLCH** for perceptually uniform lightness across themes
